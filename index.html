POC 02 - Métodos de arrays

Sort

Em Java, o método sort é utilizado para ordenar elementos em um array ou uma coleção (como uma List). O Java fornece o método sort na classe Arrays para ordenar arrays e na classe Collections para ordenar coleções.

Filter

O conceito de filter é usado para criar uma nova coleção ou stream com elementos que atendem a um critério específico. Em Java, isso pode ser feito usando a API de Streams introduzida no Java 8.

Map

O Map é usado para transformar cada elemento de uma coleção em outro elemento. Em Java, isso é feito novamente com a API de Streams.

Reduce

O reduce é usado para reduzir uma coleção a um único valor, como uma soma, multiplicação ou concatenação.

Spread

O conceito de spread (espalhamento) não existe diretamente em Java como existe em JavaScript, onde é usado para "espalhar" elementos de um array ou objeto em uma nova estrutura. No entanto, em Java, podemos usar métodos como addAll para combinar elementos de várias coleções.
