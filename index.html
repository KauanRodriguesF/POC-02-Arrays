POC 02 - Métodos de Arrays

Array Original

Array original que será utilizado em todos os exemplos: [5, 12, 8, 130, 44].

Método Sort

O método sort() ordena os elementos do array. No exemplo, o array será ordenado em ordem crescente.

Método Filter

O método filter() cria um novo array com todos os elementos que passam no teste implementado pela função fornecida. Neste caso, o filtro seleciona os números maiores que 10.

Método Map

O método map() cria um novo array com os resultados da função aplicada em cada elemento do array original. Aqui, cada número é multiplicado por 2.

Método Reduce

O método reduce() aplica uma função em um acumulador e em cada valor do array (da esquerda para a direita), resultando em um único valor. Neste exemplo, ele soma todos os números do array.

Operador Spread

O operador spread (...) permite expandir um array em elementos individuais. Aqui, ele é usado para adicionar novos números ao array original.
